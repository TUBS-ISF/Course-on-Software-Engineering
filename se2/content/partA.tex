\section{Software Evolution: Laws \& Principles}
\input{content/02a-lawsprinciples}
\lessonslearned{

	\item Lehman's laws: software is modified and gets more complex\\
	Continuing Change, Increasing Complexity, Conservation of Familiarity, Continuing Growth, Declining Quality, Feedback System
	\item Principles for software evolution: KISS, DRY, RERO
	\item Next: Concrete examples to apply principles

}
